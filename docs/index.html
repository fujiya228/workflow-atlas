◆ Workflow Atlas

Specify a project's states (nodes) × operations (flows) as JSON, and read it as one interactive HTML.

プロジェクトの 状態 (node) × 操作 (flow) を JSON で仕様化し、1枚のインタラクティブ HTML として読む。

Read a spec, don't draw a diagram. Each atlas is a single dependency-free HTML file. Open it, press 1–0 to pick an operation, ← → to step through it. Switch among 9 color schemes from the header dropdown.

図を「描く」のではなく、仕様を「読む」。各 atlas は外部依存ゼロの単一 HTML。 開いたら 1–0 で操作を選び、← → でステップを送る・戻す。右上のドロップダウンで 9 種のカラースキームを切替。
